# maxim-ANIP
Repository for ANIP 2026 paper on energy prices and EU defence spending

## Repository Structure

```
maxim-ANIP/
├── data/
│   ├── raw/        # Unmodified source data as received
│   └── clean/      # Processed and cleaned datasets
├── docs/           # Project documentation, data dictionaries, notes
└── README.md
```

See the `README.md` inside each folder for details on its contents and conventions.
